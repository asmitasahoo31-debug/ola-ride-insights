# OLA Ride Insights Dashboard 🚖

An end-to-end data analytics project analyzing ride-sharing data from OLA to extract business insights.  
The project uses **Python, MySQL, Power BI, and Streamlit** to clean, analyze, visualize, and deploy ride analytics.

---

## 📊 Project Overview

Ride-sharing platforms generate large volumes of data related to bookings, drivers, payments, and customer behavior.  
This project analyzes OLA ride data to identify patterns in:

- Ride demand
- Customer behavior
- Revenue generation
- Vehicle performance
- Ratings and cancellations

The goal is to transform raw ride data into actionable business insights using data analytics tools.
