# OLA Ride Insights Dashboard

This project is a **data analytics dashboard built using Streamlit** to analyze ride data from a ride-sharing platform similar to OLA.

## Project Overview

The dashboard provides insights into ride activity, revenue, customer ratings, and vehicle performance using interactive visualizations.

## Key Insights

* Total rides and revenue
* Booking status distribution
* Ride trends over time
* Revenue by payment method
* Vehicle type performance
* Driver vs customer ratings

## Tech Stack

* Python
* Streamlit
* Pandas
* Plotly
* GitHub
* Streamlit Cloud

## Features

* Interactive filters
* KPI metrics
* Time-series visualization
* Pie charts and bar charts
* Fully deployed web dashboard
